#### Examples

##### Image

```html
import EntypoSymbol

var symbol:EntypoSymbol = EntypoSymbol(text:EntypoIcon.install, size:25)
var iconImage:UIImage = symbol.imageWithSize(CGSizeMake(25, 25))
```

#### Installation (CocoaPods)
pod EntypoSymbol
